# Lecture Summaries for Data Mining Course

## Lecture 1: Introduction to Data Mining
- **Main Concepts:**
  - Definition of Data Mining
  - Relevance of Data Mining in various fields
  - Data Mining process

- **Key Formulas:**
  - No specific formulas for this lecture.

- **Important Definitions:**

  - Information: Processed data that provides value.

- **Exam Tips:**
  - Understand basic terminology and concepts.
  - Be prepared to explain how data mining can be applied in real-world scenarios.

## Lecture 2: Data Preprocessing
- **Main Concepts:**
  - Importance of data cleaning and preparation
  - Data transformation techniques

- **Key Formulas:**
  - Normalization formula: \[ X' = \frac{X - min(X)}{max(X) - min(X)} \]

- **Important Definitions:**
  - Data Cleaning: The process of correcting or removing inaccurate records.
  - Overfitting: When a model learns noise in the training data rather than the intended outputs.

- **Exam Tips:**
  - Focus on understanding different preprocessing methods.
  - Be familiar with examples of data transformation techniques.

## Lecture 3: Classification Algorithms
- **Main Concepts:**
  - Overview of classification
  - Common classification algorithms (e.g. Decision Trees, SVM)

- **Key Formulas:**
  - Accuracy formula: \[ Accuracy = \frac{TP + TN}{TP + TN + FP + FN} \]

- **Important Definitions:**
  - True Positive (TP), True Negative (TN), False Positive (FP), False Negative (FN)

- **Exam Tips:**
  - Understand how different algorithms work.
  - Be prepared to compare and contrast classification methods.

## Lecture 4: Clustering Techniques
- **Main Concepts:**
  - Introduction to clustering
  - Understanding K-Means and Hierarchical Clustering

- **Key Formulas:**
  - K-Means formula: \[ J = \sum_{i=1}^{K} \sum_{j=1}^{n} ||x^{(j)} - \mu_i||^2 \]

- **Important Definitions:**
  - Clusters: Groups of similar data points.
  - Centroid: The center point of a cluster.

- **Exam Tips:**
  - Know how to apply clustering algorithms.
  - Understand the differences between various clustering methods.
